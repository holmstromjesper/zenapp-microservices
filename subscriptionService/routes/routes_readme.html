ZenApp backend

  • /users

    • GET - ?userID=0, Lookup User
    • POST - Create User

  • /subscriptions

    • GET - ?userID=0 Find User subscriptions
    • POST - subscribe to service

  • /positions

    • GET - Get User Position
    • POST - Update Position

  • /services

    • GET - = ?list=[1,2,3] List Services,
    • POST - Check service criteria for subscribed services
